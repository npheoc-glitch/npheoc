# Ghana Integrated Public Health Emergency Platform (GIPHEP)
### National Outbreak Intelligence & Emergency Coordination Dashboard — Ghana

GIPHEP is an advanced, production-grade epidemiological intelligence and emergency decision-support dashboard designed for the **Ghana National Public Health Emergency Operations Centre (PHEOC)**. It unifies high-level multi-agency dashboard interfaces with dark-mode epidemiological tracking layouts used by international disease control organizations.

## Key Features
- **Zero-Backend Processing**: Direct local stream cleansing and computation of multi-indicator CSV/Excel surveillance rosters.
- **Advanced Epi-Curve Visuals**: Custom multi-layer configurations displaying rolling averages, transmission shifts, and silent-phase anchors.
- **Dynamic Risk Stratification Matrices**: Complete structural coverage across all 16 administrative regions and localized MMDAs.
- **Automated Insights Engine**: Fully integrated rule-base executing variance checks and trigger metrics across surveillance tiers.

## Deployment Instructions for Streamlit Cloud
1. Push this workspace directory directly to your GitHub repository.
2. Log in to [Streamlit Community Cloud](https://share.streamlit.io/).
3. Click **New app**, select your repository, branch, and specify `app.py` as the entrypoint file.
4. Expand **Advanced settings**, ensure Python version is set to `3.10+`, and click **Deploy**.
